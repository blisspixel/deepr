# Data Visualization

When this skill is active, you can format data as tables and charts.

## Tool usage
- **markdown_table** — Create formatted markdown tables with sorting (free)
- **ascii_chart** — Create ASCII bar charts for quick visualization (free)

## When to use
- When presenting comparative data
- When the user asks to see data in table or chart form
- When summarizing numerical results

## Response guidelines
- Use tables for exact values and detailed comparisons
- Use charts for quick visual comparisons
- Add context and interpretation alongside data displays
